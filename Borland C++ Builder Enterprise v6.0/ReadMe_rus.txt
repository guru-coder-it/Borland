Borland C++ Builder Enterprise 6.0 for Windows NT\2000\XP + Update 
Borland C++Builder - является инструментом который действительно реализовал RAD-стиль разработки приложений применительно к самым различным классам задач, включая работу с Internet, ActiveX, с проектами уровня предприятия, и все это - с использованием стандартного C++. Borland C++Builder тесно интегрирован с обеими базовыми технологиями создания распределенных систем - CORBA и COM. Borland C++Builder совместим с Visual C++. Поддерживает большинство технологий современной разработки.

Установка:
Требуется Windows NT/2000/XP
Для установки запустите bcb6.exe
Распакуйте в любую временную папку.
Устанновка запустится автоматически.

При зпросе Serial Number и Autorization Key запустите keygen\bcb6kg.EXE
Нажмите кнопку "Generate" для получения регистрационных данных.
Внимание !!! Не закрывайте генератор.
Если в конце устанновки выдастся сообщение, не волнуйтесь. Просто нажмите кнопу "Отмена".

После установки приложения запустите C++Builder.
При предложение активизации выберите пункт 
"Register or activate by phone or web browser"
Сокпируйте "Registration Code" в генератор (он у вас не закрыт ?)
Вы получитие активационный код (Activation Key)

Для установки обновления запустите Update\BCB6_ENT_Update1.exe
Не забудте удалить дистрибутивы после устанновки.
